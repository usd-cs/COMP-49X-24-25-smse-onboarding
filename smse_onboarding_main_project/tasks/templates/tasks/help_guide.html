Advanced SMSE Admin Help Guide

Learn how to navigate and manage the advanced admin page efficiently.

Overview of the Advanced Admin Page

The Advanced Admin Page is where administrators can manage faculty, users, and onboarding tasks in advanced settings.

Each section in the admin page corresponds to an area of the onboarding system.

Navigating the Advanced Admin Page
  • Dashboard: Displays available sections like Users, Groups, Tasks, and Faculty.
  • Search Bar: Quickly find records by entering names, emails, or task titles.
  • Filters: Use the right sidebar to filter by categories like status, role, or department.
Explanation of Each Link
  • Groups: Manage user permission groups (e.g., Admins, Faculty, New Hires).
  • Users: View and edit user accounts, assign permissions, and reset passwords.
  • Tasks: Manage faculty onboarding tasks, assign tasks, and track progress.
  • Faculty Documents: Upload and view faculty resumes and other documents.
  • Faculty: View all faculty members, edit details, assign onboarding tasks.
Searching & Filtering

Searching:

  • Use the search bar to find faculty members by name or email.
  • Search tasks by title to find specific onboarding tasks.

Filtering:

  • Use filters (right side panel) to show only pending tasks or incomplete onboarding.
  • Filter faculty by department, role, or onboarding status.
Changing User Roles & Permissions

To change permissions:

  • Go to Users in the admin panel.
  • Click on a user and select their permissions (Admin, Staff, Faculty).
  • Admins can add, delete, and manage all records, while staff may have limited access.
Managing Faculty & Assigning Tasks

To edit faculty details:

  • Go to Faculty in the admin panel.
  • Click on a faculty member and update their department, contact info, or job title.

To assign tasks:

  • Go to Tasks in the admin panel.
  • Click on an existing task or create a new one.
  • Select the faculty members to assign the task to.
